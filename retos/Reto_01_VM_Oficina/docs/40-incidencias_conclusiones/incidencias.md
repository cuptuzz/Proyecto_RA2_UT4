# EJERCICIO 4. Incidencias y Conclusión Final

## Incidencias encontradas
* **Problema:** El instalador de Windows 10 forzaba la creación de una cuenta Microsoft, bloqueando el avance sin internet.
* **Solución:** Se deshabilitó temporalmente el adaptador de red en VirtualBox para habilitar la opción de "Cuenta sin conexión" y crear el usuario local "Admin".
